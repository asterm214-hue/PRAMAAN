# PRAMAAN 💚
### Secure, Trauma-Informed Digital Testimony Platform

**PRAMAAN** is a state-of-the-art full-stack application designed to empower survivors of trauma by providing a safe, secure, and supportive environment to document their testimonies. Built with a "Privacy-First" and "Trauma-Informed" philosophy, PRAMAAN bridges the gap between raw personal experiences and structured legal evidence.

---

## 🌟 Key Features

### 🔐 Secure Recording & Vault
*   **Multi-Modal Testimony:** Record your story in the way that feels most comfortable—Video, Audio, or Text.
*   **End-to-End Encryption:** All testimonies are encrypted using AES-256 before being stored in your private vault.
*   **Zero-Knowledge Architecture:** Only you have the keys to your data. PRAMAAN cannot access your personal files.

### 🤖 AI-Organized Timeline
*   **Chronological Mapping:** Our AI processing engine analyzes your testimony to build a structured, chronological timeline of events.
*   **Key Fact Extraction:** Automatically identifies critical facts, witnesses, and locations that are essential for legal proceedings.
*   **Guided Support:** Trauma-informed prompts help you recall details without feeling overwhelmed.

### ⚖️ Legal Empowerment
*   **Document Generation:** Transform your testimony into structured legal drafts like Primary Victim Statements and Affidavits.
*   **Secure Sharing:** Share your encrypted documents directly with trusted lawyers, police departments, or NGOs with granular permission controls.
*   **Revocable Access:** Revoke sharing permissions at any time with a single click.

### 🎨 Premium User Experience
*   **Calm UI/UX:** A soothing, dark-mode-first aesthetic designed to reduce stress during the documentation process.
*   **Micro-Animations:** Fluid transitions and subtle feedback (powered by Framer Motion) create a living, responsive interface.
*   **Accessibility:** Designed to be intuitive and supportive across all devices.

## 🚀 Getting Started

### Prerequisites
*   Node.js (v18+)
*   Python (3.10+)

### Installation

1. **Clone the Repository:**
   ```bash
   git clone https://github.com/asterm214-hue/PRAMAAN.git
   cd PRAMAAN
   ```

2. **Frontend Setup:**
   ```bash
   npm install
   npm run build
   ```

3. **Backend Setup:**
   ```bash
   # Activate the virtual environment first (Windows)
   .venv\Scripts\activate

   # Then install dependencies
   pip install -r requirements.txt
   ```

4. **Run the Application:**
   ```bash
   # Make sure the virtual environment is active (see step 3)
   python app.py
   ```
   *Access the app at `http://localhost:8000`*
